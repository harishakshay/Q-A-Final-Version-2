# Doc Converter

A utility to convert documents (.pdf, .md, .txt) from input_docs/ to plain text files in output_txt/.

## Project Structure
- `input_docs/` — Source documents (.pdf, .md, .txt)
- `output_txt/` — Converted plain text files
- `scripts/convert_to_txt.py` — Conversion script
- `requirements.txt` — Python dependencies

## Usage
1. Place your documents in `input_docs/`.
2. Run the conversion script to process files into `output_txt/`.
3. Install dependencies: `pip install -r requirements.txt`
